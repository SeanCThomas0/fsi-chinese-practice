# FSI Standard Chinese — Speaking Practice

A zero-dependency web app for practicing **spoken Mandarin** with the public-domain
**FSI Standard Chinese: A Modular Approach** course — the U.S. Foreign Service
Institute program used to train American diplomats. All 300+ tapes and the
student texts/workbooks stream directly from
[fsi-languages.yojik.eu](https://fsi-languages.yojik.eu/).

## Features

- **Full course catalog** — Resource Module (pronunciation & romanization, numbers,
  classroom expressions, time & dates), core modules 1–9 (ORN → LIC), the six
  optional modules, review tapes, and criterion tests.
- **Drill-focused player** — A-B section looping (repeat one exchange until you can
  shadow it), playback speed 0.5×–1.25×, ±5s skip, keyboard-first controls.
- **Record & compare** — record yourself with the mic, then play the looped tape
  section followed immediately by your own recording to compare pronunciation.
- **Progress tracking** — check off tapes as you finish them; per-module progress
  is saved in your browser (localStorage).
- **PDF links** — every module's Student Text and Workbook one click away.

## Running it

The app is static — it just needs any HTTP server (mic recording requires
`localhost` or HTTPS):

```sh
python3 -m http.server 8000 -d app
# then open http://localhost:8000
```

## How to practice (the FSI way)

1. Start with the **Resource Module → Pronunciation & Romanization** tapes to train
   your ear on tones, initials, and finals.
2. Work the core modules in order. Each unit has three tape types:
   - **C** (Comprehension) — listen and understand,
   - **P** (Production) — *speak out loud* in the pauses,
   - **D** (Drill) — rapid pattern drills, also out loud.
3. Follow along in the Student Text; do the workbook exercises.
4. Use the **A-B loop** on any sentence you fumble, slow it down, record yourself,
   and compare until you match the native speaker.
5. Take the **Criterion Test** tape at the end of each module before moving on.

## Keyboard shortcuts

| Key | Action |
| --- | --- |
| `Space` | Play / pause |
| `←` / `→` | Skip back / forward 5s |
| `L` | Set loop point A, then B, then clear |
| `K` | Clear loop |
| `R` | Start / stop recording |
| `E` | Play your last recording |

## Regenerating the catalog

`app/data/catalog.json` is generated by crawling the yojik.eu directory listings.
If files change upstream:

```sh
node scripts/build-catalog.mjs
```
